# Tech Stack & Dependencies

## 1. Core Architecture
* **Framework:** Next.js 14 (App Router), React.
* **Hosting:** Vercel (Edge network for speed).
* **Styling:** Tailwind CSS (utility-first, zero config).

## 2. AI Integration
* **Provider:** Google Gemini SDK (`@google/generative-ai`).
* **Model:** `gemini-2.5-flash` (Lightning fast, cheap, perfect for generating strict JSON payloads).

## 3. UI/UX Utilities
* **Animations:** `framer-motion` (for the dramatic, math-calculating reveal of how much time they wasted).
* **Icons:** `lucide-react`.
